📄 How to Create PDFs for MDragon Data Tools

The PDFs don't exist yet! You only have markdown (.md) files. Here's the easiest way to create them:

Method 1: Google Docs (EASIEST - 5 minutes)

Step 1: Open any .md file in VS Code (e.g., probability-fundamentals.md)
Step 2: Select All (Ctrl+A) and Copy (Ctrl+C)
Step 3: Go to Google Docs and create a new document
Step 4: Paste the content (Ctrl+V)
Step 5: Click File → Download → PDF Document (.pdf)
Step 6: Save the PDF with the correct name (e.g., probability-fundamentals.pdf)
Step 7: Move the PDF to: statools/public/resources/

Repeat for all 16 markdown files!

Method 2: Word → PDF (If you have Microsoft Word)

Step 1: Open markdown file in VS Code
Step 2: Copy all content
Step 3: Paste into Microsoft Word
Step 4: File → Save As → PDF
Step 5: Save to statools/public/resources/
